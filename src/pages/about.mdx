# About

## Mission

Pedersen commitments show up constantly in modern cryptography — confidential
transactions, range proofs, voting schemes — but most explanations of them are
either terse academic papers or hand-wavy blog posts that skip the part where you
actually understand *why* the hiding and binding properties hold. This site tries
to close that gap: a plain-spoken explanation, backed by working code and live
demos you can actually poke at, aimed at engineers rather than cryptographers.

Every historical claim and citation on this site is sourced to a specific paper or
primary reference (see [History](/history) and [Further reading](/further-reading));
nothing is stated from memory without a source. Every interactive demo computes
real values in your browser — see [Interactive](/interactive) and
[Code](/code) — nothing is mocked or precomputed.

## Who runs this

This is an independent educational project, not affiliated with Torben Pedersen,
IACR, CRYPTO, or any of the other organizations and researchers whose work it
cites and links to.

## Proposing a correction or addition

The site's source is public on GitHub at
[github.com/Qubitpower/pedersen-foundation](https://github.com/Qubitpower/pedersen-foundation).
If you spot an error — a wrong date, a broken link, a math mistake in one of the
demos — please open an issue or a pull request there.
